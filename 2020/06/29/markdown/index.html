搭建GitHub博客

这篇博客详细介绍win10的系统是如何用hexo写博客以及搭建GitHub博客

准备阶段
1、下载nodeis (直接搜官网下载）
2、下载git
win10不限速下载git网址：
https://npm.taobao.org/mirrors/git-for-windows/

写新的GitHub博客

  • 创建新博客
    dengya@LAPTOP-6GPGTBM6 MINGW64 /e/blog
    $ hexo new “一篇新的博客”

  • 查看
    dengya@LAPTOP-6GPGTBM6 MINGW64 /e/blog/source/_posts
    $ dir

  • 用其他的软件去编辑.md文件，如markdown

  • 在文件blog目录下执行如下操作
    $ hexo clean
    $ hexo g
    $ hexo s （将写好的文件上传到本机）
    $ hexo d （将写好的文件上传到GitHub）
